### java.io包主要涉及文件，网络数据流，内存缓冲等的输入输出。

### Java的IO包主要关注的是从原始数据源的读取以及输出原始数据到目标媒介。
以下是最典型的数据源和目标媒介：
文件
管道
网络连接
内存缓存
System.in, System.out, System.error(注：Java标准输入、输出、错误输出)

### Java IO的用途和特征
文件访问
网络访问
内存缓存访问
线程内部通信(管道)
缓冲
过滤
解析
读写文本 (Readers / Writers)
读写基本类型数据 (long, int etc.)
读写对象
